
# Enrichissement des Quartiers des Villes Secondaires (Benin & Senegal)

## Objectif

Ajouter significativement plus de quartiers pour les villes secondaires du Benin (Parakou, Bohicon) et du Senegal (Thies, Kaolack) afin de permettre une selection d'adresse precise pour les utilisateurs de ces regions.

## Etat Actuel

| Ville | Pays | Quartiers actuels |
|-------|------|------------------|
| Parakou | Benin | 7 quartiers |
| Bohicon | Benin | 3 quartiers |
| Thies | Senegal | 7 quartiers |
| Kaolack | Senegal | 3 quartiers |

## Impact

- Les utilisateurs des villes secondaires pourront selectionner leur quartier avec precision
- Chaque quartier a des coordonnees GPS associees pour le calcul de proximite
- Les livreurs pourront mieux localiser les destinations
- Amelioration de l'experience utilisateur pour 4 villes importantes

## Structure des Donnees

Chaque quartier suit le format existant :

```text
{
  name: "Nom du Quartier",
  lat: latitude,
  lng: longitude,
  aliases: ["alias1", "alias2"],  // Variantes orthographiques
  region: "Ville Parente",         // Ville a laquelle appartient le quartier
  type: "neighborhood"             // Type = quartier
}
```
