
Le lien idéal pour le bouton **"Commence ici"** dans votre modèle WhatsApp est celui qui redirige l'utilisateur directement vers son tableau de bord, sur l'onglet des amis, avec la fenêtre d'ajout d'un nouvel ami déjà ouverte.

### URL à utiliser :
**`https://joiedevivre-africa.com/dashboard?tab=amis&add=true`**

### Pourquoi ce lien est le meilleur ?
- **Direct et contextuel** : Il amène l'utilisateur exactement là où il doit agir (Cercle d'amis).
- **Gain de temps** : Le paramètre `&add=true` déclenche automatiquement l'ouverture du formulaire pour ajouter un ami, évitant à l'utilisateur de chercher le bouton "+" sur la page.
- **Support multi-appareils** : Ce lien fonctionne aussi bien sur mobile (Deep Linking vers l'application installée ou le site web) que sur ordinateur.

### Instructions pour Meta Business Manager :
Dans votre écran de création de modèle (vu dans votre capture d'écran) :
1. **Type d'action** : Conserver "Consulter le site Web"
2. **Texte du bouton** : "Commence ici" (comme vous l'avez déjà écrit)
3. **Type d'URL** : Statique
4. **URL du site Web** : Copiez et collez `https://joiedevivre-africa.com/dashboard?tab=amis&add=true`

*Note : Si vous souhaitez un lien plus court et plus simple sans ouvrir le formulaire automatiquement, vous pouvez utiliser `https://joiedevivre-africa.com/dashboard?tab=amis`.*
